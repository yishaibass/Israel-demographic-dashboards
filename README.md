# Israel Dashboards

Three interactive Hebrew data dashboards with a shared home page.

- **balance-sheet/** — Israeli household balance sheet (CBS longitudinal survey 2012–2023)
- **voting-simulator/** — Knesset mandate simulator
- **fiscal-position/** — Fiscal net position of the three sectors, 2018→2023

Home page: `index.html`. Each dashboard has a "🏠 דף הבית" button linking back home.
